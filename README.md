# BIM Segurança Eletrônica

**Protector Sistemas** — Ferramenta de projeto para segurança eletrônica e infraestrutura tecnológica.

## Sobre

BIM (Building Information Modeling) voltado para projetos de segurança eletrônica: CFTV, controle de acesso, alarmes, redes e infraestrutura. Permite projetar, validar conexões e gerar lista de materiais (BOM) completa.

## Funcionalidades

- **Canvas interativo** — Drag-and-drop de dispositivos com zoom e pan
- **Paleta de dispositivos** — Câmeras, controle de acesso, alarmes, rede, infraestrutura
- **Engine de conexões** — Validação automática de compatibilidade entre dispositivos e cabos
- **Nobreaks configuráveis** — AC e DC com catálogo de modelos reais (Intelbras, MCM)
- **Rack como container** — Arraste equipamentos para dentro do rack com gestão de U
- **Automação** — Leitor facial/LPR/Tag → Motor/Quadro via Borne SAK
- **BOM completa** — Dispositivos, cabos com metragem, baterias, acessórios de rack
- **Múltiplos cenários** — Residencial, comercial, condomínio, industrial
- **Topologia em árvore** — Visualização da hierarquia de rede
- **Validação em tempo real** — Alertas de conexões inválidas e dispositivos desconectados

## Stack Atual

Single HTML file com React 18 + Babel (standalone). Roadmap para migração Next.js + Supabase.
